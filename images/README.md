## How to Add Images

1. See `IMAGE_SOURCES.md` for where to find images for each home
2. Download images and rename them with the correct pattern
3. Place in this folder
4. Images will automatically load in the app

## Image Requirements

- Format: JPG (preferred) or PNG
- Size: < 500KB each (compress if larger)
- Resolution: 1200px width recommended
- Naming: lowercase, hyphens, numbered 1-10
